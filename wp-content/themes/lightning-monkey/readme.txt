== DESCRIPTION ==

Lightning Monkey is a fast, responsive theme with enough options and page layouts to keep anyone happy.

Lightning Monkey also has a built in landing page or home page template.

== INSTALLATION ==

Manual installation:
1. Upload the `lightning-monkey` folder to the `/wp-content/themes/` directory

Installation using "Add New Theme"
1. From the Admin Dashboard, use the menu to select Themes -> Add New
2. Search for 'Lightning Monkey'
3. Click 'Install' which will open the theme's listing
4. Click 'Install'

Activation and Use
1. In the 'Themes' menu, activate the theme
2. See Appearance -> Theme Settings for Theme Settings
3. See Appearance -> Customize Home Page for Home Page Settings
